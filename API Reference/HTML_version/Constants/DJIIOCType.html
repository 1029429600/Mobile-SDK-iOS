DJIIOCTypeCourseLock

As the aircraft follows a specific route, the direction is locked. The method lockCourseUsingCurrentDirectionWithResult: will lock the direction of the aircraft to the current direction.

Declared In DJIIOCMission.h.

DJIIOCTypeHomeLock

Having the aircraft in this mode will allow the aircraft to move towards the home point without having to know the exact direction the aircraft needs to be in to return to home. For example, if the aircraft is extremely far out from the home point, using the remote controller to move the aircraft backwards will move the aircraft towards home instead of backwards.
